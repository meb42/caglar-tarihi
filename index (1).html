Gemini API Anahtarı
Çağlar Tarihi · v4 · Ücretsiz
Oyunu oynamak için ücretsiz bir Google Gemini API anahtarı gerekiyor. Tamamen ücretsiz, kredi kartı gerekmez. Bir kere gir, her zaman hatırlanır.
Nasıl Alınır? (Ücretsiz)
1. aistudio.google.com/apikey adresine git
2. Google hesabınla giriş yap (Gmail yeterli)
3. "Create API Key" butonuna bas
4. Key'i kopyala, aşağıya yapıştır
Key sadece senin cihazında saklanır. Google'ın ücretsiz limitinde her gün 1500 istek hakkın var.
